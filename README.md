# Section 4 : Basic RGB LED Button Control

This Arduino project controls a **common-anode RGB LED** using a single push button.  
When the button is pressed, the LED turns ON; when released, it turns OFF.  
The LED and button states are managed using simple digital I/O logic.

## Modes
| Mode | Name | Description |
|------|------|--------------|
| 1 | **Constant ON** | LED stays lit continuously |
| 2 | **Slow Blink** | LED blinks once every second |
| 3 | **Medium Blink** | LED blinks twice per second |
